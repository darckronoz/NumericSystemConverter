Numeric system converter! v1.0

numeric system converter, this converts any number on any base to any number on any base. only from base 2 to base 36.

v2 Update: Now you can convert float point numbers on any numeric system. **for numbers with float point part large it only takes the first 4 numbers, So when converting not exact numbers from base to base the answer isn't exact.

example: 898.69 on base 10 -> 2422.4554 on base 7 and 2422.4554 on base 7 -> 898.6837
